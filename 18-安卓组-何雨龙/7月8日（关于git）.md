> #### 使用Git与GitHub
>>1. 使用Git Bash对项目库进行各项操作:  
>>>     (1)init git 初始化一个本地库  
>>>     (2)git clone xxx 克隆库文件至本地与github配合使用
>>>     (3)git add xxx 添加文件
>>>     (4)git commit -m"xxx" 为各种操作进行备注
>>>     (5)git push 提交更改至GitHub所对应clone到本地的库
>>2. 注意事项：
>>>     (1)使用Git bash时要给Git bash定位，不定位的话本地文件默认存在C盘  
>>>     (2)想用Gitbash对克隆的库进行操作:在clone网页时务必先fork
